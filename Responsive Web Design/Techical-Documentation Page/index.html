Introduction

JavaScript is a versatile, high-level programming language widely used in web development.

It allows developers to build interactive websites and applications.

JavaScript is primarily used for front-end web development but can also be used for back-end.

console.log("Welcome to JavaScript!");
  • It is a dynamically typed language.
  • JavaScript is interpreted and executed in the browser.
Syntax

JavaScript has a C-like syntax, making it familiar to developers with experience in languages like C++ or Java.

Variables are defined using `var`, `let`, or `const`.

let x = 10;
  • JavaScript is case-sensitive.
  • Semicolons are optional but recommended.
Data Types

JavaScript supports both primitive and complex data types.

Primitive types include strings, numbers, and booleans.

let str = "Hello, World!";
  • Primitive types: String, Number, Boolean, Undefined, Null, Symbol, BigInt.
  • Objects and arrays are reference types.
Functions

Functions allow code reuse and modular programming in JavaScript.

They can be declared using the `function` keyword.

function add(a, b) { return a + b; }
  • Functions can return values.
  • They can be assigned to variables.
Objects

Objects store key-value pairs and can hold functions as methods.

They are fundamental in JavaScript for organizing data.

let user = { name: "John", age: 30 };
  • Objects are created using curly braces `{}`.
  • They can contain nested objects and arrays.
Events

Events allow JavaScript to respond to user interactions.

Common events include `click`, `mouseover`, and `keydown`.

document.getElementById("btn").onclick = function() { alert("Clicked!"); };
  • Event listeners can be added using `addEventListener()`.
  • Events can be triggered manually using `dispatchEvent()`.
